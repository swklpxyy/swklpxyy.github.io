간단한 사용자 전환

Windows XP를 사용하면 한 대의 컴퓨터를 여러 사용자가 간편하게 공유해서 사용할 수 있습니다.

컴퓨터를 사용하는 각 사용자는 암호로 보호되는 별도의 계정을 만들어 사용자 개인 설정을 만들고 개인적인 파일을 사용할 수 있습니다. 컴퓨터에서 여러 계정을 동시에 활성화시킬 수 있으며 신속하고 간편하게 사용자 전환이 가능합니다.

예를 들어, 여러분이 작업하는 동안 동료가 전자 메일을 확인하고 싶다고 합니다. 이럴 경우 간단히 동료의 사용자 계정으로 전환하기만 하면 됩니다. 여러분은 작업 중인 프로그램을 닫거나 로그오프할 필요는 없습니다. 또한 동료는 작업하는 동안 여러분의 파일을 볼 수 없으며 여러분의 계정으로 다시 전환하더라도 동료의 파일을 볼 수 없게 됩니다. 여러분의 사용자 계정으로 전환하면 작업 화면이 동료의 사용자 계정으로 전환하기 이전과 똑같은 상태로 돌아가기 때문입니다.
