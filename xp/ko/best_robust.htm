향상된 성능, 안정성, 호환성

Windows XP Professional은 엔터프라이즈 수준의 성능과 안정성에 전례가 없는 사용자 편의성을 결합하여 기업용 소프트웨어의 새 표준을 세웠습니다. Microsoft의 검증된 Windows 2000 기술을 토대로 개발된 Windows XP Professional에는 Microsoft Windows XP Home Edition의 모든 기능이 포함되어 있으며 특히 기업 및 고급 사용자를 위한 향상된 새 기능이 추가되었습니다.

시작 시간이 빨라졌으며 기업용 프로그램이 보다 안정적으로 실행됩니다.

작업 파일을 늘여서 여러 모니터를 통해 한 번에 볼 수 있으며, 이중 인터페이스 디스플레이 어댑터가 장착된 컴퓨터에서 서로 다른 두 개의 프로그램을 동시에 볼 수도 있습니다.

Microsoft ClearType™ 기술을 사용하면 특히 랩톱 컴퓨터에서 화면 내용을 더욱 쉽게 읽을 수 있습니다.

Windows XP Professional은 다국어 기능이 지원되므로 세계 어느 곳에서나 사용할 수 있습니다. 24개 언어 중에서 선택할 수 있는 것은 서로 다른 언어를 구사하는 다국적 직원들이 같은 컴퓨터를 공유하는 다국적 기업에서는 아주 유용한 기능입니다.

랩톱 컴퓨터를 사용하세요? Windows XP의 향상된 전원 관리 기능은 배터리 수명을 연장시킵니다. 뿐만 아니라 랩톱 컴퓨터와 다른 PC를 함께 사용하면 컴퓨터의 전력 소모 방식을 보다 효율적으로 관리할 수 있습니다.
