안전하고 쉬운 컴퓨터 작업

Windows XP는 여러분이 컴퓨터를 쉽고 즐겁게 사용할 수 있도록 합니다! 역동적인 작업 처리, 뛰어난 성능, 멋진 디자인, 풍부한 도움말 등이 여러분을 기다리고 있습니다. 이 모든 장점을 두루 갖춘 Windows XP의 신뢰성과 보안성 역시 타의 추종을 불허합니다.

디지털 미디어의 세계로 초대

집과 사무실에서 인터넷을 통해 사진, 음악, 비디오, 컴퓨터 게임과 같은 신나는 디지털 미디어의 세계를 경험하고 즐겨 보세요.

가정과 사무실의 연결
