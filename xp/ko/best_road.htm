언제, 어디서나 연결 가능

전세계 어디에서든지 사무실 컴퓨터의 전자 메일을 읽고 파일을 보고 프로그램을 실행시켜 보세요. Windows XP Professional의 진보된 통신 기능을 사용하면 최신의 이동 컴퓨팅과 첨단 무선 기술을 마음대로 통제할 수 있습니다.

원격 데스크톱을 사용하면 Windows XP를 실행하는 컴퓨터에서 인터넷을 통해 Windows XP Professional을 실행하는 컴퓨터의 화면을 볼 수 있습니다. 이것은 랩톱 컴퓨터, 집에 있는 컴퓨터 또는 지구 반대편에 있는 고객의 사무실 컴퓨터에서 "회사 업무 처리"가 가능해 진다는 의미입니다. 마치 동시에 서로 다른 두 장소에 있는 것과도 같은 셈이죠!

Windows XP의 보안 무선 연결을 통해 음성, 비디오 및 인스턴트 메시징을 사용하면 실시간으로 통신 및 공동 작업이 가능합니다. Windows XP가 설치된 다른 무선 장치가 통신 범위 내에 있을 때마다 자동으로 여러분에게 알려 주므로 인터넷에 연결하느라 시간을 낭비할 필요도 없습니다.

Windows XP Professional에서는 인터넷 또는 다른 파일 공유 연결이 동시에 10개까지 지원됩니다. 네트워크 브리징을 사용하면 무선, 이더넷 및 집 전화선을 통해 동시에 통신을 할 수도 있습니다.

따라서 Windows XP Professional을 사용하면 이동 컴퓨팅이 더 이상 복잡하지 않습니다. 여러분은 언제 어디서나 인터넷에 연결되어 있으니까요!
