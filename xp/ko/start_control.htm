제어판

Windows XP의 제어판을 사용하면 간편하게 컴퓨터 설정을 사용자 지정할 수 있습니다. 시작 메뉴에서 제어판을 클릭하면 열리는 제어판에서는 다음과 같은 작업을 수행할 수 있습니다.

  • Windows XP 디자인과 작업 방식 변경
  • 프로그램 또는 하드웨어 장치의 추가 또는 제거
  • 네트워크 연결 및 사용자 계정 설정
  • 이 밖에도 더 있습니다!
