在旅途中及在全世界

從世界任何地方在您的辦公室電腦上閱讀電子郵件、檢視檔案，及執行程式。Windows XP Professional 的進階通訊功能可讓您掌控最新行動運算及最先進的無線技術。

在網際網路上，從任何執行 Windows XP 的其它電腦，使用「遠端桌面」檢視執行 Windows XP Professional 之電腦的螢幕。這表示您可以從膝上型電腦、家用電腦，或在地球另一端的客戶辦公室「到達辦公室」。就像同時在兩地！

Windows XP 中的安全無線連線可讓您使用語音、視訊，及立即訊息，在事情發生的同時進行即時通訊與合作。無需浪費時間去嘗試接通，每當其它配有 Windows XP 的無線裝置在範圍中時，您就會自動收到通知。

Windows XP Professional 允許同時有多達 10 個網際網路或其它檔案共用連線。網路橋接允許在無線、Ethernet，及家用電話線上的通訊同時進行。

有了 Windows XP Professional，行動通訊變得輕而易舉。您將總是保持連線狀態。
